# johncanola10.github.io

Personal portfolio site for Jonathan Canola.

Live site: <https://johncanola10.github.io>

## Overview

This is a static portfolio site built without a framework or build step. The goal was to keep the presentation direct: a short personal introduction, a few selected projects, a simple skills section, and clear contact links.

## Stack

- HTML
- CSS
- GitHub Pages

## Local Preview

Because the site is static, you can open `index.html` directly in a browser or serve it locally with any simple static server.

For example:

```bash
python3 -m http.server
```

Then visit `http://localhost:8000`.

## Design Goals

- no framework overhead
- no build tooling
- fast load time
- readable in a few seconds
- easy to edit by hand
